# Nueramarcos/mlx

Personal fork of [ml-explore/mlx](https://github.com/ml-explore/mlx).

MLX primarily targets Apple Silicon. This fork tracks Linux-side experiments and validation work.

## Smoke check

```bash
git clone https://github.com/Nueramarcos/mlx.git
cd mlx
python3 -c "import pathlib; assert pathlib.Path('mlx').is_dir(); print('ok')"
```
